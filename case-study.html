The objective of The Hive is to establish a marketable platform in-order to create a more efficient system for their B2B sales channels, enabling them to offer access to experiences and events to their B2B partners' clients. We can bifurcate the broader categories involved in this project into two: The Hive Team (The Hive Admin & The Hive Manager) and Agency Team (Agency Admin & Agent). Our primary focus is on creating a robust web application tailored for B2B partners like travel agencies, concierge companies, other event companies, credit cards, UHNW banks, etc. The B2B partners can create sub profiles for each agent to submit requests regarding bookings for their clients. Additionally, we have built web applications for Super Admin and The Hive Manager who will be responsible for overseeing supplier relationships for each event, maintain an up-to-date event calendar, handle follow-ups, and distribute content to our B2B partners, among other functions.

Creating a platform that seamlessly integrates The Hive's B2B sales channels requires careful consideration and a well-thought-out process.
Here are some of the points we focused on, while working on this platform.
  • User-Centric Design:

Emphasize a user-friendly interface with easy login and prominent Event Calendar for intuitive event browsing.

  • Request Submission:

Simplify the "Request to Book" process with a customizable form capturing key details like quantity, date, budget, and preferences.

  • Customized Proposal Creation:

Enable The Hive Manager to generate tailored proposals efficiently, facilitating communication with the Super Admin for approvals.

  • Payment and Processing:

Implement secure payment options directly from clients or indirectly through agents at the NET rate, automating booking confirmation and commission payments post-event for transparency and efficiency.
